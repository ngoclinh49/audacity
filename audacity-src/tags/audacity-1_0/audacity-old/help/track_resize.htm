Vertical Track Resizing

For horizontal resizing, which is called zooming, visit the View Menu page.

Vertically resizing a track is fairly simple.

  • Click and hold the mouse button on the resizing handle (see figure),
  • move mouse up or down untill the track has the wanted vertical size
  • release mouse button

Note:

Stereo tracks are handled like two mono tracks, so each channel of a stereo track will have its own resize handle.
