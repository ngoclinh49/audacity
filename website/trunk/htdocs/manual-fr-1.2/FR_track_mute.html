Bouton Muet

Cliquer sur ce bouton pour passer la piste en mode Muet, sans effacer la piste. Cette action ne peut être annulée avec la fonction Défaire, mais il suffit simplement de cliquer à nouveau dessus pour annuler l'action précédente.

En mode Muet, les pistes ayant le bouton activé ne seront pas jouées.  (Bouton en gris Ombré).

Pour désactiver le mode Muet, cliquez à nouveau sur le bouton. (qui redevient gris Clair)

Cette étape de mode solo n'est pas sauvegardée dans le fichier projet et n'affecte pas l'Export ou le Mixage.
