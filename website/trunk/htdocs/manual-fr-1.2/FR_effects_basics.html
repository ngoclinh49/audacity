Comment les effets fonctionnent dans Audacity?

1. Sélectionnez une partie d'audio

Utilisez la souris ou les  raccourcis clavier pour sélectionner un peu d'audio. Vous pouvez pré écouter cette sélection autant de fois que vous souhaitez et ajuster les bornes de cette sélection.

2. Sélectionner un effet

Vous pouvez appliquer tout effet d'Audacity ou un effet plug-in VST (Windows & Mac) ou LADSPA(Linux).

3. Améliorer les paramètres que vous voulez

Jouer en boucle, klaxonner .... Faites ce que vous voulez sans en parler aux autres.

4. Laisser le logiciel travailler

Enfoncez le bouton OK. Ou Annuler si vous n'êtes pas sûr de quelque chose...

5. Pas satisfait ? ANNULEZ

Si le résultat ne vous plait pas alors :

Cliquez dans le menu Edition sur Annuler (ou pressez CTRL+Z).

Puis revenez à l'étape 3. vous pourrez ainsi annuler indéfiniment.
